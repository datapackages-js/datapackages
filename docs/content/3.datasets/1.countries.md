# Countries

The `@datapackages/countries` package provides a comprehensive list of countries, including their codes, names, and other metadata.

## Installation

```bash
npm install @datapackages/countries
```

## Usage

```typescript
import { CountryService } from "@datapackages/countries";

// Initialize the service
const service = CountryService.getInstance();
await service.init();

// Access data
const data = await service.getAll();
// Output: [{ cca2: "US", name: { common: "United States", ... }, ... }, ...]

// Search (using @datapackages/plugin-search)
const results = await service.search("France");
```

## Features

- **ISO Codes**: Includes ISO 3166-1 alpha-2 and alpha-3 codes.
- **Multilingual**: Supports localized names (if configured).
- **Type-Safe**: Full TypeScript definitions for the country object.
