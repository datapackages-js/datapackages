# Medical Terminologies

The `@datapackages/medical-terminologies` package enables access to standard medical coding systems like CIM-10 (ICD-10) and CCAM.

## Installation

```bash
npm install @datapackages/medical-terminologies
```

## Usage

```typescript
import { France } from "@datapackages/medical-terminologies";

const { Profession } = France;
const service = new Profession();

// Initialize
await service.init();

// Find a code
const data = await service.getAll();
const item = data.find(i => i.code === "A00");
```
